Для кого подойдет

  • Для профессиональных фотографов, кто хочет научиться фотографировать новорожденных

    Отточите навыки работы с фотографией, научитесь делать ретушь на продвинутом уровне и разберётесь в трендах. Выйдете на новый уровень и сможете больше зарабатывать.

  • Для начинающих фотографов, кто хочет специализироваться в ньюборн фотографии

    Изучите основы профессиональной фотографии: как выстраивать кадр, проводить съёмки и обрабатывать снимки. Сможете делать крутые фотографии в качестве хобби или начать творческую карьеру.

  • Для мам новорожденных, кто хочет научиться профессионально фотографировать своего малыша

    Научитесь делать профессиональные фотографии своего малыша.

Чему вы научитесь

  • укладывать малыша в красивые позиции, не разбудив

  • обрабатывать фотографию всего за 5 минут с помощью Photoshop (для фотографов) и отдельный урок я создала для мам о том, как обработать фото малыша с помощью специальных приложений для телефона

  • управлять светом для получения сочных снимков

  • Дополнительный урок для мам о том, как профессионально обработать фото малыша с помощью телефона
